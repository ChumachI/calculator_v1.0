SmartCalc_v1
Implementation an extended version of the usual calculator, which can be found in the standard applications of each operating system in the C programming language using structured programming. Besides calculating expressions, it also supports the use of the _x_ variable and the graphing of the corresponding function. In addition to the basic part, loan and deposit calculators are also implemented.
